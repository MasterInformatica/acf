\begin{frame}
\frametitle{Definición}
El uso de Internet y las redes de comunicaciones han facilitado la
creación de un nuevo comercio llamado \textbf{comercio electrónico}. Este se
basa en la prestación de servicios (venta, disposición de efectivo,
... ) ligados a una transacción financiera de forma electrónica.

Esta transacción significa un movimiento real de fondos entre el
usuario y el prestador del servicio, lo que ha supuesto la creación de
sistemas de pago seguros y eficaces. A estos sistemas de pago es lo
que llamamos \textbf{sistemas de pago electrónico}.

Un sistema de pago electrónico debe cumplir tres requisitos mínimos: 
\begin{itemize}
\item Privacidad a dicha comunicación evitando las escuchas de terceros.
\item Autentificación del usuario e integridad del mensaje. 
\item Protección que garantice al usuario que ha pagado por un servicio.
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{Estándares ISO}
La función principal de la ISO es buscar la estandarización de normas de productos y seguridad para las empresas u organizaciones a nivel internacional.
\begin{itemize}
\item ISO 7816, EMV y GSM. Para promocionar la interoperabilidad entre las tarjetas inteligentes con contacto y los lectores.
\item ISO/IEC 14443, ISO 15693. El uso de las tarjetas inteligentes sin contacto.
\item ISO 8583, Estándar para Transacciones Financieras con Mensajes originados en una tarjeta.
\item ISO 20027 La normativa Europay Mastercard Visa (EMV) es un acuerdo realizado en Europa y permite que los pagos tengan un estándar específico con estas dos tarjetas de crédito.
\item ISO 20022 para los pagos masivos en euros
\item ISO 9001 para sus principales procesos de negocio
\item ISO 27001 seguridad de la información.
\end{itemize}
\end{frame}
